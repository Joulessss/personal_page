## Resumen

<div className="overview-text">

La compatibilidad entre compuestos antioxidantes (AC) y materiales de pared (WM) es uno de los aspectos más críticos del
proceso de encapsulación, ya que la estabilidad de los compuestos encapsulados depende de la afinidad química entre ellos.
Se construyó un modelo de compatibilidad mediante aprendizaje automático (ML) para identificar combinaciones óptimas sin
recurrir a experimentos costosos y prolongados. Se recopilaron atributos de los componentes antioxidantes y de pared, y se
construyeron dos conjuntos de datos. Luego se realizó un proceso de vinculación para conectar ambos conjuntos e identificar
relaciones significativas entre parámetros de AC y WM, necesarias para definir compatibilidad o incompatibilidad y para
enriquecer el conjunto de datos con pares señuelo. Posteriormente, un análisis estadístico examinó las correlaciones entre
variables y se aplicó un Análisis de Componentes Principales (PCA) para reducir la dimensionalidad sin perder información
esencial. Se utilizó el algoritmo KNN para abordar la clasificación de combinaciones compatibles e incompatibles. El modelo
alcanzó una exactitud de 0.92, con sensibilidad de 0.84 y especificidad de 1, evidenciando un desempeño robusto al clasificar
correctamente combinaciones positivas y negativas.

</div>

## Contribución

- Compilación e integración de datos fisicoquímicos de AC y WM, incluyendo la generación de pares señuelo (incompatibles).
- Identificación de relaciones clave AC–WM mediante análisis de correlación y reducción de variables con PCA.
- Entrenamiento y evaluación de un clasificador KNN para predecir compatibilidad, reportando exactitud, sensibilidad y especificidad.

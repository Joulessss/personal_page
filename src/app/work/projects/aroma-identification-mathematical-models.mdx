## Overview

This project implements aroma pattern identification on sample data from `.nos` files using mathematical and data-driven models.

## What I Built

- Data ingestion and preprocessing pipeline for `.nos` sensor outputs.
- Mathematical modeling workflow for aroma feature extraction and classification.
- Reproducible notebook implementation for analysis and experimentation.
